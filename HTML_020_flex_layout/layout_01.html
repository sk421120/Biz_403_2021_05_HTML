box model tag를 float로 설정을 하면 box width에 따라 화면에서 한 라인에 여러 tag를 배열할 수 있다

그런 float로 만든 layout은 floating되기를 원하기 않은 tag에 까지 영향을 미치게 된다

display 속성

block -> inline

inline -> block

div tag에 display를 inline으로 설정했더니 갑자기 block 설질을 버리고 한 라인데 여러개의 box를 배열할 수 있게 되었다

그런데 inline 속성은 box의 width를 설정할 수 없다.

다행히 inline 속에서 block의 성질을 포함한 inline-block 속성이 추가된다
